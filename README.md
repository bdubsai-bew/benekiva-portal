# Benekiva AI Claims Platform

AI-powered claims processing with document intelligence, fraud detection, automated underwriting, and auto-adjudication.

## 🚀 Live Demo

**[benekiva-portal.vercel.app](https://benekiva-portal.vercel.app)**

## Features

- **AI Claims Intake** — Guided wizard with animated auto-fill from document extraction
- **Document Intelligence** — Multi-step AI extraction with OCR, classification, and data mapping
- **Claims Dashboard** — Pipeline view with stage tracking, processing time savings, and AI agent activity
- **Fraud Detection** — Real-time document verification and anomaly detection
- **Automated Underwriting** — AI-driven risk assessment and policy evaluation
- **Auto-Adjudication** — Rules engine + AI for straight-through claim processing
- **Analytics** — Claims volume, processing times, and AI performance metrics
- **AI Agent Activity Feed** — Live feed showing agent actions across the platform

## Tech Stack

- **Framework:** Next.js 16 (App Router, Turbopack)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + shadcn/ui
- **Charts:** Recharts
- **Icons:** Lucide React
- **Deployment:** Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages

| Route | Description |
|-------|-------------|
| `/login` | Authentication screen |
| `/dashboard` | Main dashboard with AI activity feed |
| `/claims` | Claims list with pipeline view |
| `/claims/new` | New claim intake wizard |
| `/document-intelligence` | Document upload & AI extraction |
| `/analytics` | Performance analytics |
| `/settings` | Platform settings |
